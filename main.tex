\section{Background}

\subsection{Discrete Fourier Transform}
The Discrete Fourier Transform is a fundamental tool used in digital signal processing that 
enables sophisticated analysis and synthesis of discrete-time signals \cite{lyons2010understanding}.   
The DFT transforms a signal from the time domain into its frequency-domain representation,  
allowing the signal to be expressed as a sum of sinusoids \cite{lyons2010understanding}. Mathematically, the DFT is defined as: 
\begin{align}
  X[m] = \sum_{n=0}^{N-1} x[n] \, e^{-j 2\pi n\frac{m}{N}}
\end{align}
where $N$ is the number of samples from the input signal $x[n]$, $n$ denotes the time-domain index, $m$ represents the frequency bin index, 
$n$ is the index of the input sequenc in the time-domain and $j = \sqrt{-1}$. Each coeffecient in $X[m]$ is complex-valued and can be 
expressed in terms of magnitude and phase, which describe the contribution of a corresponding frequency component to the overall signal \cite{mcfee_dstbook}.

\subsection{Windowing}
Windowing is a function used to isolate parts of a discrete time signal so that analysis on the signal can be performed on contiguous blocks \cite{barry2008time}.
More specifically, the input signal is multiplied by a windowing function whose domain is nonzero for a particular range. This results in a small localized snapshot of the input signal,
and every other value outside of the windows domain is reduced to zero \cite{royer2019pitch}. Figure 1 below demonstrates the functionality of a window function.

\subsection{Short-Time Fourier Transform}
The Short-Time Fourier Transform (STFT) is a time-frequency representation used to compute the DFT of contiguous windowed segments of a signal. 
This is accomplished by multiplying the signal by a windowing function so that only a small portion of the signal is nonzero \cite{dolson1986phase}. 
Mathematically, it is similar to equation $(1)$ but with a few modifications: the addition of a windowing function, the domain of the input signal, 
and how the output is interpreted. Below is the STFT: 
\begin{align}
  X[m,k] = \sum_{n=0}^{N-1} x[n+kH]w[n]e^{-2\pi jmn/N}
\end{align}
Where $X[m, k]$ is the 2-dimensional output array indexed by both frequency bin and time frame and $x[n + kH]$ is the section of the input signal 
determined by frame index and hop size \cite{royer2019pitch}. The outcome of the STFT is a representation that transforms a signal from the time 
domain to the frequency domain. In this form, the signal's frequency components can be systematically modified and recombined to reconstruct a new signal.   


\section{Related Works}

Pitch-shifting has been approached by a number of time--frequency methods. 
Early methods, such as Dolson's (1986), employed the use of the Short-Time 
Fourier Transform in conjunction with the phase vocoder algorithm. 
In particular, Dolson describes a pitch-shifting method that relies on 
time-scale manipulation to modify the pitch of a given signal \cite{dolson1986phase}. 
For example, by stretching a signal, making it longer in the time domain, 
and then rescaling it back to its original length, the overall frequency 
content of the signal is effectively decreased, which corresponds to 
lowering the signal's pitch. Dolson acknowledges, however, that this 
approach is imperfect: specific constraints must be enforced when 
performing a pitch shift, as failing to do so may introduce unwanted 
artifacts into the signal or result in a reconstructed signal that has 
lost information \cite{dolson1986phase}.

Another limitation of the standard phase-vocoder technique, as discussed 
in \cite{laroche1999new}, is that only a single linear pitch-scale modification 
can be applied during processing. That is, all frequency components are 
multiplied by the same pitch-shifting factor. Consequently, generating harmonized 
outputs requires repeated application of the full algorithm \cite{laroche1999new}. 
This significantly increases computational cost and limits real-time 
feasibility.

These challenges lead to further innovations in the phase vocoder algorithm. 
Modern implementations shift away from the standard two-step timescale and resample 
approach and adopt an analysis and resynthesis paradigm \cite{degotzen2000traditional}. 
The new approach leverages phase information obtained from the STFT step to more accurately 
reconstruct the modified signal\cite{ degotzen2000traditional}.
